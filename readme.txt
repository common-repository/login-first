This plugin makes users login first before they can see your website. Real handy when you are developing a site or just want to to shield it from search engines.

== Description ==

What you can expect from this plugin:

* When this plugin is active, your website can not be crawled or visited without loggin in first.
* When you are running an environment that needs to be shielded from the outside, this could be the solution.
* When this plugin is deactivated it does not leave any file or data behind.
* Clean, fast and safe solution for redirecting your visitors to the standard Wordpress login screen.

== Installation ==

Upload the the plugin (ZIP) to your website, activate it and you're done. No extra need for optional settings or complicated stuff.
